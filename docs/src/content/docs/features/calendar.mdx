title: Calendar
description: Month-view calendar sidebar
---

The calendar sidebar provides a visual month grid for navigating daily notes.

## Features

- Click any day to open or create a daily note
- Dot indicators show which days have existing notes
- Navigate between months with arrow buttons
- Configurable week start day (Sunday, Monday, or Saturday)
- Optional ISO week numbers

## Opening the calendar

- Click the **calendar icon** in the left ribbon
- Or use the command: **Daily Notes NG: Open calendar**
