---
title: Natural language dates
description: Inline date parsing with @-trigger
---

Type a trigger character (default `@`) followed by a natural language date expression to get inline date suggestions.

## Examples

- `@today` → `[[2026-03-23]]`
- `@next friday` → `[[2026-03-28]]`
- `@in 3 days` → `[[2026-03-26]]`
- `@last month` → `[[2026-02-23]]`

## Settings

- **Enable** — Toggle the feature on/off
- **Trigger character** — Character that activates suggestions (default: `@`)
- **Date format** — Output format for inserted dates
- **Insert as link** — Wrap in `[[wikilinks]]` (links to daily notes)
