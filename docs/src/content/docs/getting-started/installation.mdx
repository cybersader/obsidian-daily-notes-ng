## Manual Installation

1. Download `main.js`, `manifest.json`, and `styles.css` from the [latest release](https://github.com/cybersader/obsidian-daily-notes-ng/releases)
2. Create the plugin folder: `<your-vault>/.obsidian/plugins/daily-notes-ng/`
3. Copy the three downloaded files into that folder
4. Restart Obsidian
5. Go to **Settings > Community Plugins** and enable "Daily Notes NG"

## Using BRAT

If you use the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat):

1. Open BRAT settings
2. Click "Add Beta Plugin"
3. Enter: `cybersader/obsidian-daily-notes-ng`
4. Enable the plugin

## Migration from Periodic Notes

If you're currently using the Periodic Notes plugin, Daily Notes NG can import your settings automatically. See the [migration guide](/obsidian-daily-notes-ng/guides/migration/) for details.
