title: Quick start
description: Get up and running with Daily Notes NG in 5 minutes
---

## 1. Enable daily notes

After installing the plugin, go to **Settings > Daily Notes NG**. Daily notes are enabled by default with these settings:

- **Folder**: `Journal/Daily`
- **Format**: `YYYY-MM-DD`
- **Template**: (none)

## 2. Open the calendar

Click the calendar icon in the left ribbon, or use the command palette: **Daily Notes NG: Open calendar**.

## 3. Create today's note

Click today's date in the calendar, or use the command: **Daily Notes NG: Open today's daily note**.

## 4. Set up a template (optional)

Create a template file (e.g., `Templates/Daily Note Template.md`) and set it in the plugin settings under **Daily > Template**.
