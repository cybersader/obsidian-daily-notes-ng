---
title: Migration from Periodic Notes
description: How to migrate from the abandoned Periodic Notes plugin
---

If you're currently using the Periodic Notes plugin by Liam Cain, Daily Notes NG can import your existing settings.

## Automatic import

1. Install and enable Daily Notes NG
2. If Periodic Notes is detected, the plugin will offer to import its settings
3. Your folder paths, date formats, and template paths will be preserved

## Manual migration

1. Note your current Periodic Notes settings (folder, format, template for each periodicity)
2. Enter the same values in Daily Notes NG settings
3. Disable the old Periodic Notes plugin

## Compatibility

- Your existing periodic notes files are not modified
- Other plugins that use `obsidian-daily-notes-interface` will detect Daily Notes NG settings
- Calendar clicking behavior works the same way
