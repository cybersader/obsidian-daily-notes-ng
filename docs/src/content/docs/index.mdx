## Why Daily Notes NG?

The Obsidian daily notes ecosystem has a **maintenance crisis**. [Periodic Notes](https://github.com/liamcain/obsidian-periodic-notes) and [Calendar](https://github.com/liamcain/obsidian-calendar-plugin) have been abandoned for ~4 years with 151+ open issues. Users are forced to combine 5+ plugins for a complete daily notes workflow.

Daily Notes NG consolidates the best of **Periodic Notes**, **Calendar**, **Rollover Daily Todos**, **Update Frontmatter Modified Date**, **Natural Language Dates**, and the **Journal plugin** into one actively maintained solution — plus a **multi-user identity system** for shared vaults that no other daily notes plugin offers.

## Features at a glance

- **[Periodic notes](/obsidian-daily-notes-ng/features/periodic-notes/)** — Daily, weekly, monthly, quarterly, and yearly notes with independent folder/format/template configs
- **[Calendar sidebar](/obsidian-daily-notes-ng/features/calendar/)** — Month-view calendar with dot indicators for existing notes
- **[Template integration](/obsidian-daily-notes-ng/features/templates/)** — Built-in variable resolution with seamless Templater bridge
- **[Todo rollover](/obsidian-daily-notes-ng/features/todo-rollover/)** — Carry incomplete checkboxes from yesterday's note
- **[Frontmatter dates](/obsidian-daily-notes-ng/features/frontmatter-dates/)** — Auto-populate created and modified dates with smart debouncing
- **[Natural language dates](/obsidian-daily-notes-ng/features/natural-language-dates/)** — Type `@next friday` to insert date links inline
- **[Multi-user identity](/obsidian-daily-notes-ng/features/identity/)** — Per-person folders, creator attribution, device registration, group notes

## For developers

Daily Notes NG is built with **Bun**, **TypeScript**, and **esbuild**. It includes a full [development guide](/obsidian-daily-notes-ng/development/setup/), [agentic testing workflows](/obsidian-daily-notes-ng/development/testing/), and a separate [test fixtures plugin](/obsidian-daily-notes-ng/development/test-fixtures/) for rapid iteration.
