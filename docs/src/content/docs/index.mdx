## Features

- **Periodic notes** - Daily, weekly, monthly, quarterly, and yearly notes with independent configurations
- **Calendar sidebar** - Month-view calendar with dot indicators for existing notes
- **Template integration** - Built-in variable resolution with seamless Templater bridge
- **Todo rollover** - Automatically carry incomplete todos from yesterday's note
- **Frontmatter dates** - Auto-populate created and modified dates with smart debouncing
- **Natural language dates** - Type @next friday to insert date links inline

## Why Daily Notes NG?

The Obsidian daily notes ecosystem has a **maintenance crisis**. Periodic Notes and Calendar have been abandoned for ~4 years. Daily Notes NG consolidates the best features from 6+ plugins into one actively maintained solution.
