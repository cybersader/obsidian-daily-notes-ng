All commands are available via the Command Palette (`Ctrl/Cmd + P`).

## Navigation

| Command | Description |
|---------|-------------|
| Open today's daily note | Opens or creates today's daily note |
| Open previous daily note | Navigate to the previous day |
| Open next daily note | Navigate to the next day |
| Open previous weekly note | Navigate to the previous week |
| Open next weekly note | Navigate to the next week |

*(Similar commands exist for monthly, quarterly, and yearly)*

## Calendar

| Command | Description |
|---------|-------------|
| Open calendar | Opens the calendar sidebar |
