All settings are accessed via **Settings > Daily Notes NG**.

## Periodic notes

| Setting | Default | Description |
|---------|---------|-------------|
| Enable daily | `true` | Create daily notes |
| Daily folder | `Journal/Daily` | Storage folder |
| Daily format | `YYYY-MM-DD` | Filename format |
| Enable weekly | `false` | Create weekly notes |
| Weekly folder | `Journal/Weekly` | Storage folder |
| Weekly format | `gggg-[W]WW` | Filename format |

*(Monthly, quarterly, and yearly follow the same pattern)*

## Calendar

| Setting | Default | Description |
|---------|---------|-------------|
| Open on startup | `false` | Auto-open calendar sidebar |
| Show week numbers | `true` | Display ISO week numbers |
| Week start | `Monday` | First day of week |
| Dot indicators | `true` | Show dots for days with notes |

## Frontmatter dates

| Setting | Default | Description |
|---------|---------|-------------|
| Track dates | `true` | Enable auto date tracking |
| Created key | `date created` | Frontmatter property name |
| Modified key | `date modified` | Frontmatter property name |
| Date format | ISO 8601 | Moment.js format string |
| Delay | `10000` | Debounce in milliseconds |

## Natural language dates

| Setting | Default | Description |
|---------|---------|-------------|
| Enable | `true` | Enable NLP date suggestions |
| Trigger character | `@` | Character that starts suggestions |
| Date format | `YYYY-MM-DD` | Output format |
| Insert as link | `true` | Wrap in wikilinks |
